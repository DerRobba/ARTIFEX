# ARTIFEX
ARTIFEX is an Open-Source 3D-Printer like the 200 or the DuEnder. We aim at making 3D-printing simple, cheap and accesible to everyone. While usage is very simple with our AI-Model-Search and -Generator Features as well as our AI-Slicing-Features, ARTIFEX can be used in a more professional enviroment for people who want to learn.
